# Senza Misura Reborn

Piattaforma digitale premium per preservare e rendere accessibile l'eredità storica di Senza Misura.

## Root progetto

Tutto il lavoro pesante e i media sono su:

`E:\SENZA_MISURA_REBORN`

## Comandi

```powershell
python tools/generate_content_map.py
python -m http.server 4173
```

Poi apri:

`http://localhost:4173`

## File chiave

- `content-map.md` — inventario completo delle risorse indicizzate.
- `data/media-inventory.json` — inventario tecnico.
- `data/media-library.json` — metadata usati dal Media Center.
- `public/media/` — media migrati e archiviati.
- `src/app.js` — componenti e interazioni del sito.
- `src/styles.css` — design system cinematico.

## Principio di preservazione

If any content exists only on the legacy websites, it must be migrated, preserved, indexed and made accessible from the new platform. No historical audio, video, teaching, document or downloadable resource may be lost during migration.
